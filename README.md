# Grep Light

A simple grep-like program written in Rust.

The motivation behind developing this utility is purely educational. Being a new
Rustling, I find learning the language, its features, and idiomatic design
easiest by doing.

## Features

The following crates were used in this project:

- [clap] - command line argument parser
- [regex] - regular expression engine

[clap]: https://docs.rs/clap/4.5.4/clap/index.html
[regex]: https://docs.rs/regex/latest/regex/
